# CSIT213-Java-Programming-Assignments

This repository is created to **upload Java programming assignment codes** for backup and to allow **students to reference the code**.  
Each branch corresponds to a specific assignment, making it easy to access the examples you need.

---

📁 **How to Access Assignment Codes**

1. Navigate to the repository.  
2. Click on the **branch dropdown menu** (top-left).  
3. Select the branch for the assignment you want to view:  
   - `Assignment-1` → Code examples for Assignment 1  
   - `Assignment-2` → Code examples for Assignment 2  
   - `Assignment-3` → Code examples for Assignment 3  
   *(and so on for each assignment)*

---

📂 **Opening the Files**

- It is recommended to use **Apache NetBeans** to open and run the Java files.  
- Simply clone the repository, switch to the desired assignment branch, and open the project in NetBeans.

### Steps:
1. **Download/Clone the Repository**
   - Option A: Click the green **Code** button → Download ZIP → Extract the files.  
   - Option B: Use Git:  
     ```bash
     git clone https://github.com/your-username/CSIT213-Java-Programming-Assignments-Code.git
     ```

2. **Open NetBeans**
   - Launch **Apache NetBeans** on your computer.

3. **Import the Project**
   - Go to **File → Open Project**.  
   - Browse to the folder where you extracted/cloned the repository.  
   - Select the project folder and click **Open**.

4. **Run the Code**
   - Right-click the main file or project.  
   - Choose **Run File** or **Run Project** to execute the program.

---
